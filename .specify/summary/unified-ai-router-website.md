# Unified AI Router Website - Project Summary

## Project Overview

This specification package creates a developer-focused website for the Unified AI Router, providing documentation and landing page for developers implementing an OpenAI-compatible router with automatic fallback functionality.

## What's Included

### Constitution

**File**: [`.specify/memory/constitution.md`](.specify/memory/constitution.md)

Core principles guiding the project development.

## Key Goals

- **Developer Education**: Help developers implement OpenAI-compatible router with automatic fallback
- **Quick Onboarding**: Enable developers to get started in under 5 minutes
- **Comprehensive Documentation**: Provide configuration and API reference materials

## Technology Stack

- **Framework**: VitePress (Markdown-based static site generator)
- **Deployment**: Static site hosting (GitHub Pages, Vercel, Netlify)

## Implementation

The website uses VitePress for documentation. Existing docs/ folder contains the implementation.
